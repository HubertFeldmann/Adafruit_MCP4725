Sets the output voltage to a fraction of source vref. (Value can be 0..4095)

Parameters
[in]outputThe 12-bit value representing the relationship between the DAC's input voltage and its output voltage.
[in]writeEEPROMIf this value is true, 'output' will also be written to the MCP4725's internal non-volatile memory, meaning that the DAC will retain the current voltage output after power-down or reset.
i2c_frequencyWhat we should set the I2C clock to when writing to the DAC, defaults to 400 KHz
Returns
True if able to write the value over I2C
